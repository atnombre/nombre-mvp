## Why Archived?

These files were reorganized on January 24, 2026 to:

1. **Consolidate** - Combine duplicate information into single authoritative sources
2. **Improve navigation** - Clear folder structure with descriptive names
3. **Human-friendly** - Documentation readable by both developers and AI agents
4. **Maintainable** - Easier to keep one source of truth updated
